# CPP_Arcade

**Subject [here](https://github.com/thibautcornolti/Arcade/blob/master/Subject.pdf)**

### Usage

```
$> make
$> ./arcade
```

* Arrow up & down to change the game
* Arrow left & right to change the graphical library
